# Blockchain Based Management for Organ Donation and Transplantation

## 📌 Project Overview

Blockchain Based Management for Organ Donation and Transplantation is a web-based application designed to improve transparency, security, and trust in the organ donation and transplantation process.

The system leverages cryptographic hashing and blockchain concepts to securely maintain donor, patient, and transplantation records. By storing records in an immutable manner, the application helps prevent data tampering and ensures integrity throughout the organ donation lifecycle.

---

## 🎯 Objectives

- Secure management of organ donor records.
- Maintain transparency in organ allocation and transplantation.
- Prevent unauthorized modification of sensitive data.
- Improve trust between donors, recipients, hospitals, and administrators.
- Utilize blockchain-inspired hashing mechanisms for record verification.

---

## ✨ Features

### Admin Module
- Manage donor and patient information.
- Authorize users.
- Monitor organ donation requests.
- View transplantation records.
- Verify records through blockchain hashing.

### Donor Module
- Register and login securely.
- Add organ donation details.
- View donation status.
- Track organ donation records.

### Patient Module
- Register and login.
- Request organ transplantation.
- Track transplantation requests.
- View request status.

### Blockchain Security
- Cryptographic hash generation.
- Data integrity verification.
- Secure record storage.
- Tamper detection mechanism.
